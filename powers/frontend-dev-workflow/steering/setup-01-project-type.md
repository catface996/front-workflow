# 检测模块：项目类型检测

> 检测项目当前状态

## 检测方式

检查以下文件/目录是否存在：
- `package.json`
- `src/` 目录

## 判断结果

| 类型 | 判断条件 | 后续流程 |
|------|---------|---------|
| 空项目 | 无 `package.json` 或仅有基础配置 | 完整初始化：技术栈 → 目录 → 设计系统 → 骨架 |
| 已有项目 | 存在 `package.json` + `src/` | 检测现有配置 → 补充缺失项 |
